리스트 : 요소가 여러 개 들어있는 목록
  음수 사용 가능
  리스트 안에 리스트 사용 가능
  리스트 접근 연산자를 이중으로 사용 가능

리스트 연산자
  연결 (+) : 리스트 두개 이상 연결
  반복 (*) : 리스트 1개 반복
  len() : 길이

리스트 요소 추가
  append() : 리스트 뒤에 요소 추가
  insert() : 위치를 지정하고 그곳에 요소 추가
  extend() : 한 번에 여러 개의 요소 추가

인덱스로 요소 제거
  del : 특정 요소 하나 제거
    del에 특정한 범위를 입력 시 그 부분 모두 제거
  pop : 괄호 안에 아무 인덱스 입력도 하지 않으면 마지막을 제거
  값으로 제거할 경우 remove 사용
  모두 제거는 clear

리스트 정렬 : sort
  오름차순은 그대로
  내림차수는 reverse=True

내부에 있는지 확인 : in/not in
