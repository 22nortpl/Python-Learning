튜플 : 리스트와 비슷, 그러나 결정된 요소 변경 불가
람다 : 함수를 간단하고 쉽게 선언
  함수의 매개변수로 함수 전달 : 콜백 함수

파일 처리 : 텍스트 파일과 바이너리 파일로 분류
  파일 열기 : 파일 읽기 또는 파일 쓰기
  open() 함수 사용
    w : write, 새로 쓰기 모드
    a : append, 뒤에 이어서 쓰기 모드
    r : read, 읽기 모드
  닫을 때는 close() 사용

제너레이터 : 파이썬의 특수한 문법 구조
  yield를 사용하며 함수 내부의 코드가 실행되지 않음

스택 : first in last out 꼴
힙 : 객체 자료형들이 저장되어 있는 거대한 창고

함수 값 복사 : 기본 자료형은 값은 그대로 유지
  전역 변수가 함수 변수의 영향을 받지 않는 현상
  그러나, 리스트는 주소로 값을 복사
