모듈 : 여러 변수와 함수 존재
  표준 : 파이썬 내에 내장
  외부 : 다른 사람들이 만들어서 공개

math 모듈 : 수학적 요소
구문 사용법:
  from 구문 : 가져오고 싶은 변수 또는 함수에 여러개 입력 가능
    ex : from math import sin, cos,...
  as 구문 : 이름 충돌 발생 시 사용
    ex : import math as m

random 모듈 : 무작위 생성

os 모듈 : 운영체제와 관련된 기능을 가진 모듈

datetime 모듈 : 날짜를 출력하는 다양한 방법을 가진 모듈

time 모듈 : 시간과 간련된 기능

urllib 모듈 : url을 다루는 라이브러리, 네트워크의 자원위치 확인 시 사용

operator.itemgetter() 함수 : key매개변수에 itemgetter 함수를 저장
