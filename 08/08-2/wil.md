isinstance(인스턴스, 클래스) : 그 객체가 어떤 클래스로부터 만들어졌는지 확인

str 이외에 특수한 이름의 함수
  eq : equal(같다)
  ne : not equal(같지 않다)
  gt : greater than(크다)
  ge : greater than or equal(크거나 같다)
  lt : less than(작다)
  le : less than or equal(작거나 같다)

클래스 변수 : class 구문 바로 아래의 단계에 변수 선언
예시:
  class 클래스 이름:
    클래스 변수 = 값

접근법:
  클래스 이름.변수 이름
