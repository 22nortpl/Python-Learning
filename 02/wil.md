자료형 : 자료의 형태
기본 자료형
  문자열 : 메일 제목, 메시지 내용
  숫자 : 물건의 가격, 학생의 성적
  불 : 친구의 로그인 상태
  
이때 이스케이프 문자를 사용해서 문자열 생성 가능

여러 줄 문자열을 만들 때
1. \n 사용
2. """ 사용
혹여나 마지막에 의도치 않은 줄바꿈을 방지 : """\을 사용

문자열 연산자
연결 : +
반복 : *
선택(인덱싱) : []
  0부터 시작된다. 만약 -1부터 시작할 경우, 순서는 거꾸로 간다.

문자 범위 슬라이싱 [:] 사용
첫 번째 숫자 : 포함, 두 번째 숫자 : 포함하지 않음

숫자 연산자
  사칙 : +, -, *, /
  정수 나눗셈 : //
  정수 나머지 : %
  제곱 연산자 : **

변수 : 값 저장
