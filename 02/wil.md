자료형 : 자료의 형태
기본 자료형
  문자열 : 메일 제목, 메시지 내용
  숫자 : 물건의 가격, 학생의 성적
  불 : 친구의 로그인 상태
  
이때 이스케이프 문자를 사용해서 문자열 생성 가능

여러 줄 문자열을 만들 때
1. \n 사용
2. """ 사용
혹여나 마지막에 의도치 않은 줄바꿈을 방지 : """\을 사용

문자열 연산자
연결 : +
반복 : *
선택(인덱싱) : []
  0부터 시작된다. 만약 -1부터 시작할 경우, 순서는 거꾸로 간다.

문자 범위 슬라이싱 [:] 사용
첫 번째 숫자 : 포함, 두 번째 숫자 : 포함하지 않음

숫자 연산자
  사칙 : +, -, *, /
  정수 나눗셈 : //
  정수 나머지 : %
  제곱 연산자 : **

변수 : 값 저장

format() 함수 : 숫자를 문자열로 변환
다양한 형태
  d : 정수
  f : 실수
  +/- : 기호 추가
  정수 추가 : 칸 벌리기
  0이 추가되면 0으로 채우기
  .1, .2 : 소수점

대소문자 바꾸기 : upper와 lower
문자열 양옆의 공백 제거하기 : strip()
구성 파악 : .is@@()
  isalnum() : 알파벳 또는 숫자
  isalpha() : 알파벳
  isidentifier() : 식별자
  isdecimal() : 정수
  isdigit() : 숫자 인식 형태
  isspace() : 공백으로만 구성
  islower : 소문자로만 구성
  isupper : 대문자로만 구성

find() : 처음 등장 위치 찾기(왼쪽부터)
rfind() : 처음 등장 위치 찾기(오른쪽부터)

문자열과 in 연산자 : true 혹은 false로 안에 있는지 여부 확인

문자열 자르기 : split()
