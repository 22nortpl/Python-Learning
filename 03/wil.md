불 : 명제의 true 혹은 false 로 값 표현
  비교 연산자
    == : 같다
    != : 다르다
    < : 작다
    <= : 이하이다
    > : 크다.
    >= : 이상이다.

  논리 연산자
    not : 아니다
    and : 둘다 모두 true
    or : 둘 중 하나라도 true

if 구문 : 조건 만족 시 사용
else 구문 : if 조건문 뒤에 활용
elif 구문 : if 에서 예외인 것들 중에 적용되는 조건에 사용
항상 false : 0, 빈 문자열
pass : 골격을 잡고 내부 내용을 나중에 구현할 때 사용
