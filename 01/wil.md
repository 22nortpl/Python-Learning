코드 실행기 사용 명령어
  현재 폴더 확인 : is 명령어
  폴더 이동 : cd 명령어, 상위 폴더는 ..를 추가함.

문장 : 실행할 수 있는 코드의 최소 단위
프로그램 : 문장들의 모임
표현식 : 숫자, 수식, 문자열을 조합해서 값을 만들어내는 코드

키워드 : 이미 만들어진 특별한 의미의 단어
